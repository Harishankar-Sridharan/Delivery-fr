# Gérer les tickets

Le système de gestion des tickets de mission dans Nomadia Delivery simplifie le processus de gestion des demandes liées aux missions ou aux colis. Il permet aux prestataires d’ouvrir des tickets directement depuis l’éditeur de mission via l’

onglet Aide. Cela garantit que tout problème ou toute préoccupation survenant pendant une mission est signalé rapidement et résolu efficacement.

Pour accéder au module de tickets, les utilisateurs doivent disposer des droits appropriés accordés par les administrateurs
