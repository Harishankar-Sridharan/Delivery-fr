# Gestion des sous-traitants

La fonctionnalité de gestion des sous-traitants permet aux transporteurs de déléguer des tâches logistiques à des sous-traitants, ce qui fluidifie les collaborations et accroît la flexibilité opérationnelle.

Cette fonctionnalité garantit une gestion efficace des sous-traitants afin que les opérations logistiques restent optimisées et alignées sur les objectifs commerciaux.

Dans l’application Nomadia Delivery, les sous-traitants doivent être enregistrés par le transporteur avant de pouvoir télécharger les données de mission. Pour cela, le transporteur doit ajouter le sous-traitant via le module Sous-traitant.
