# Créer un utilisateur sous-traitant

Utilisez les étapes suivantes pour configurer un nouvel utilisateur sous-traitant.

Les transporteurs peuvent attribuer des utilisateurs à chaque sous-traitant afin de gérer plus efficacement les tâches de planification et d’exécution. Cette fonctionnalité permet de créer des planificateurs et des conducteurs dédiés pour chaque sous-traitant, garantissant une gestion fluide et efficace des opérations logistiques.

1. Ouvrez l’application Nomadia Delivery et accédez à Configuration > Gérer les utilisateurs.

<figure><img src="../.gitbook/assets/image (47).png" alt=""><figcaption></figcaption></figure>

2. Cliquez sur le bouton Action et sélectionnez Ajouter.

<figure><img src="../.gitbook/assets/image (48).png" alt=""><figcaption></figcaption></figure>

3. Cliquez sur OK pour créer un nouvel utilisateur.

<figure><img src="../.gitbook/assets/image (49).png" alt=""><figcaption></figcaption></figure>

Vous pouvez également créer un utilisateur à partir d’un utilisateur existant. Cela accélère le processus en copiant tous les paramètres de l’utilisateur existant vers le nouvel utilisateur.

4. Dans l’onglet Généralités, saisissez le profil, l’identifiant de connexion et le nom de famille de l’utilisateur, puis সকտիվez le statut de l’utilisateur. Ensuite, accédez à l’onglet Accès.

<figure><img src="../.gitbook/assets/image (50).png" alt=""><figcaption></figcaption></figure>

5. Activez l’accès web pour l’utilisateur et sélectionnez le site web du sous-traitant dans la liste déroulante.

<figure><img src="../.gitbook/assets/image (51).png" alt=""><figcaption></figcaption></figure>

6. Sélectionnez le bon sous-traitant (créé dans le module Sous-traitants) à associer à cet utilisateur, puis passez à l’onglet Rôles et droits.

**Remarque**: Si aucun sous-traitant n’apparaît dans la liste, assurez-vous qu’un a été créé dans le module Sous-traitants.

7. Le système attribue automatiquement les droits en fonction de l’utilisateur.

<figure><img src="../.gitbook/assets/image (52).png" alt=""><figcaption></figcaption></figure>

8. Enregistrez le nouvel utilisateur. Le sous-traitant recevra une notification par e-mail avec les instructions pour créer un mot de passe et activer le compte.

<figure><img src="../.gitbook/assets/image (53).png" alt=""><figcaption></figcaption></figure>

<br>

9. Le sous-traitant doit :
   * Cliquez sur le lien de vérification dans l’e-mail.
   * Saisissez le code de vérification reçu par e-mail et cliquez sur **Vérifier le code**.
   * Cliquez sur **Continuer**.
   * Saisissez et confirmez un nouveau mot de passe, puis cliquez sur **Continuer**.
   * Acceptez le contrat de licence de l’utilisateur final, puis cliquez sur **Continuer**.
