# Configurer l’affichage des statuts des missions

Cette fonctionnalité permet aux utilisateurs de personnaliser les statuts de mission affichés sur le tableau de bord de leur page d'accueil. En sélectionnant jusqu'à cinq statuts, les utilisateurs peuvent adapter l'affichage pour montrer les informations les plus pertinentes en fonction de leurs besoins opérationnels, ce qui les aide à rester concentrés et efficaces.

Depuis la **Page d'accueil**,

1. Cliquez sur le **Icône de compte** situé dans le coin supérieur droit de l'écran.
2. Dans le menu déroulant, sélectionnez **Mes préférences**.
3. Choisissez jusqu'à cinq statuts des statistiques de mission à afficher sur la page d'accueil parmi les options suivantes : **En attente, Non reçu, Non livré, À livrer** et **Chargée**
4. Après avoir sélectionné les statuts souhaités, cliquez sur **Enregistrer**.

5. Votre page d'accueil affichera les KPI sélectionnés.
