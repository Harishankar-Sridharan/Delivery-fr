# Mobile - Rôles et droits

| <p><br>Rôles et droits</p>           | Description                                                                                                               |
| ------------------------------------ | ------------------------------------------------------------------------------------------------------------------------- |
| Livraison                            | Lorsqu’elle est activée, les utilisateurs peuvent effectuer des opérations liées à la livraison.                          |
| Docking                              | Lorsqu’elle est activée, les utilisateurs peuvent accéder aux activités de docking et les gérer.                          |
| Chargement / déchargement            | Lorsqu’elle est activée, les utilisateurs peuvent charger et décharger des colis dans leurs véhicules.                    |
| Chargement sans préparation complète | Lorsqu’elle est activée, les utilisateurs peuvent charger des colis même si la mission n’est pas entièrement préparée.    |
| Préparer                             | Lorsqu’elle est activée, les utilisateurs peuvent préparer les missions avant leur exécution.                             |
| Réception                            | Lorsqu’elle est activée, les utilisateurs peuvent recevoir des colis à la destination ou au dépôt.                        |
| Stockage                             | Lorsqu’elle est activée, les utilisateurs peuvent déplacer des colis vers des emplacements de stockage.                   |
| Scanner les missions inconnues       | Lorsqu’elle est activée, les utilisateurs peuvent scanner et traiter des missions qui ne sont pas prédéfinies.            |
| Créer des itinéraires                | Lorsqu’elle est activée, les utilisateurs peuvent créer et gérer des itinéraires de livraison.                            |
| Gérer les livraisons non attribuées  | Lorsqu’elle est activée, les utilisateurs peuvent gérer les livraisons non attribuées à un itinéraire ou à une ressource. |
| Gérer les enlèvements non attribués  | Lorsqu’elle est activée, les utilisateurs peuvent gérer les enlèvements non attribués à un itinéraire ou à une ressource. |
| Réattribuer des missions             | Lorsqu’elle est activée, les utilisateurs peuvent réattribuer les missions à différents itinéraires ou ressources.        |
| Scan non obligatoire                 | Lorsqu’elle est activée, le scan des colis est facultatif pendant les opérations.                                         |
| Repositionner les adresses           | Lorsqu’elle est activée, les utilisateurs peuvent modifier ou corriger les adresses des missions.                         |
| Écran du superviseur                 | Lorsqu’elle est activée, les utilisateurs peuvent accéder à l’écran de supervision.                                       |
| Transfert de colis                   | Lorsqu’elle est activée, les utilisateurs peuvent transférer des colis entre des missions ou des conteneurs.              |
| Regrouper dans un conteneur          | Lorsqu’elle est activée, les utilisateurs peuvent regrouper plusieurs colis dans un seul conteneur.                       |
| Modifier les missions                | Lorsqu’elle est activée, les utilisateurs peuvent modifier les détails des missions.                                      |
| Rendre l’appel obligatoire           | Lorsqu’elle est activée, les utilisateurs doivent passer un appel avant de terminer la mission.                           |

**Politique de mot de passe pour l’accès mobile**

Le mot de passe doit contenir au minimum 8 caractères, dont au moins une lettre majuscule, une lettre minuscule et un chiffre.
