# Créer une mission de retrait urgente

Cette fonctionnalité permet d’insérer une mission de ramassage imprévue dans un itinéraire existant et publié, alors que le chauffeur est déjà sur le terrain. Elle est spécifiquement conçue pour les scénarios de ramassage, tels qu’un client demandant le retour immédiat d’un article endommagé.

1. Créer un **Nouvelle mission de ramassage**

<figure><img src="../.gitbook/assets/image (234).png" alt=""><figcaption></figcaption></figure>

2. Sélectionnez le **Mission** dans le
3. Cliquez sur le **Actions** menu de la liste.
4. Sélectionnez **Ramassage d’urgence**.

<figure><img src="../.gitbook/assets/image (235).png" alt=""><figcaption></figcaption></figure>

5. Choisissez le point d’insertion du chauffeur :

**Attribuer après**: ajoute le ramassage comme prochain arrêt.

<figure><img src="../.gitbook/assets/image (236).png" alt=""><figcaption></figcaption></figure>

6. Attribuer à **la fin**: ajoute le ramassage à la fin de l’itinéraire actuel.
7. Cliquez sur **Remplacer**

Ces missions regroupées seront exécutées une seule fois sur l’application mobile, ce qui réduit considérablement le temps d’exécution sur le terrain. De plus, même si les missions ne sont pas regroupées au préalable, l’application mobile regroupe automatiquement les missions qui partagent les mêmes informations d’adresse.
