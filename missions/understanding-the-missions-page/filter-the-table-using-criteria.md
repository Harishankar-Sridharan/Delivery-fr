# Filtrer le tableau à l’aide de critères

| Type de données    | Opérateur                                                                                                       |
| ------------------ | --------------------------------------------------------------------------------------------------------------- |
| **Alphanumérique** | <p>=<br>≠<br>Contient<br>Ne contient pas<br>Commence par<br>Se termine par<br>Est vide<br>N'est pas vide</p>    |
| **Date**           | <p>=<br>=D+<br>=D-<br>≤D+<br>≤D-<br>≥ D+<br>≥ D-<br>&#x3C;br>≤<br>><br>≥<br>≠<br>Est vide<br>N'est pas vide</p> |
| **Arithmétique**   | <p>=<br>≠<br>&#x3C;br>≤<br>><br>≥</p>                                                                           |

1. Cliquez sur le **Champ de saisie du filtre.**
2. Sélectionnez la condition appropriée parmi les suggestions.
3. Choisissez l'opérateur approprié dans le menu déroulant.
4. Saisissez la valeur requise pour la condition.
5. Appuyez sur la **Entrée** touche pour appliquer le filtre.
